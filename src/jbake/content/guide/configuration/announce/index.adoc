
= Announce
:jbake-type:   page
:jbake-status: published

Releases can be announced to several media once they have been successfully created.

[source,yaml]
[subs="+macros"]
----
# [optional]
announce:
  # Enable or disable all configured announcers.
  # Defaults to `true`.
  # [optional]
  enabled: true
----
