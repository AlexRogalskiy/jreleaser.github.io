
= Configuration
:jbake-type:   page
:jbake-status: published

`jreleaser init --format yml` or start from scratch. Other configuration formats besides YAML may be supported as well,
such as JSON, the <<_maven,Maven>> DSL, the <<_gradle,Gradle>> DSL.
