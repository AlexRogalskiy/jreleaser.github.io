= Hooks

Execute external tools before or after a JReleaser step.

== Configuration

include::partial$legend.adoc[]

[tabs]
====
YAML::
+
--
[source,yaml]
[subs="+macros"]
----
# icon:dot-circle[]
hooks:
  # Enable or disable all configured hooks.
  # Supported values are [`NEVER`, `ALWAYS`, `RELEASE`, `SNAPSHOT`].
  # Defaults to `ALWAYS`.
  # icon:dot-circle[] icon:eye-slash[]
  active: ALWAYS
----
--
TOML::
+
--
[source,toml]
[subs="+macros"]
----
# icon:dot-circle[]
[hooks]

  # Enable or disable all configured hooks.
  # Supported values are [`NEVER`, `ALWAYS`, `RELEASE`, `SNAPSHOT`].
  # Defaults to `ALWAYS`.
  # icon:dot-circle[] icon:eye-slash[]
  active = "ALWAYS"
----
--
JSON::
+
--
[source,json]
[subs="+macros"]
----
{
  // icon:dot-circle[]
  "hooks": {

    // Supported values are [`NEVER`, `ALWAYS`, `RELEASE`, `SNAPSHOT`].

    "active": "ALWAYS"
  }
}
----

--
[source,xml]
[subs="+macros,verbatim"]

<jreleaser>
