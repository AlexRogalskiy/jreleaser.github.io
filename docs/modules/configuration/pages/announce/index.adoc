= Announce

Releases can be announced to several media once they have been successfully created.

include::partial$legend.adoc[]

[tabs]
====
YAML::
+
--
[source,yaml]
[subs="+macros"]
----
# icon:dot-circle[]
announce:
  # Enable or disable all configured announcers.
  # Defaults to `true`.
  # icon:dot-circle[]
  enabled: true
----
--
TOML::
+
--
[source,toml]
[subs="+macros"]
----
# icon:dot-circle[]
[announce]

  # Enable or disable all configured announcers.
  # Defaults to `true`.
  # icon:dot-circle[]
  enable = true
----
--
JSON::
+
--
[source,json]
[subs="+macros"]
----
{
  // icon:dot-circle[]
  "announce": {
    // Enable or disable all configured announcers.
    // Defaults to `true`.
    // icon:dot-circle[]
    "enabled": true
  }
}
----
--
Maven::
+
--
[source,xml]
[subs="verbatim,+macros"]
----
<jreleaser>
  <!--
    icon:dot-circle[]
  -->
  <announce>
    <!--
      Enable or disable all configured announcers.
      Defaults to `true`.
      icon:dot-circle[]
    -->
    <enabled>true</enabled>
  </announce>
</jreleaser>
----
--
Gradle::
+
--
[source,groovy]
[subs="+macros"]
----
jreleaser {
  // icon;dot-circle[]
  announce {
    // Enable or disable all configured announcers.
    // Defaults to `true`.
    // icon:dot-circle[]
    enabled = true
  }
}
----
--
